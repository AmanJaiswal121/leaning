Introduction to HTML :

Html stands for hyper text markup language. It is a language of web which is used to create website.

In our tutorial, every topic is given step-by-step so that you can learn it in a very easy way. If you are new in learning HTML, then you can learn HTML from basic to a professional level and after learning HTML with CSS and JavaScript you will be able to create your own interactive and dynamic website. But Now We will focus on HTML only in this tutorial.

The major points of HTML are given below:

  • HTML stands for HyperText Markup Language.
  • HTML is used to create web pages and web applications
  • HTML is widely used language on the web.
  • We can create a static website by HTML only.

A basic HTML page start with :

<! Doctype html> - Specifies this is an html 5 doc.

<html> - root of an HTML page open.

<head> - contains page metadata (information about some already existing data).

<title> - title open.

</title> - title colose.

</head> - head colose.

<body> - the main body of the page open.

</body> - the main body of the page close.

</html> - root of an HTML page open.

  • We can use "insert element" or "view page source" option from chrome to look into website HTML code.

    • HTML element = start tag + content + end tag.

Comment in HTML :

Comments in html are used to mark text whic should not be parsed. Browser read whatever we written in our code.

< !--HTML comments-- >

Basic Html tags :

01. Anchor tag - it is used, whenever we need any link or any URL in our website then we use anchor tag.

eg. <a href="URL/link"content </a>.

  • In anhor tag opening and closing name should be same.
  • We can either use single (' ') or double courts (" ") in attributes.

02. Heading tag - it is used to write headings in html. It start with h1 and end with h6, it works in the manner most important to least important headings.

eg. <h1> most imp. tag </h1>.
<h2> least imp. tag </h2>.
<h3> another imp. tag </h3>.
<h4> another imp. tag </h4>.
<h5> another imp. tag </h5>.
<h6> least imp. tag </h6>.

03. Paragraph tag - it is used to add links to an existing content inside an html page.

eg.<p> paragraph </p>.

04. Relative link tag - if we want to go one html file to another html file in our own envoirment then we use this.
NOTE : it only works on same operating system.

eg.<a href="/html file name.file extension">name</a>.

05. Image tag - it is used to add any image in an html page.

eg.

    Want image online
  • <img src="image link">
    • Want image from own system
  • <img src="system path">

If we lode any image and the link or path are wrong or the image was not there, then we use alt.

eg.<img src="path/link" alt="what we wnat to show insted of that image">

06. Bold, Italic & Underline tag - they are use to hghlight the text as follows.

Bold - eg.<b> name </b>.
Italic - eg.<i> name </i>.
Underline - eg.<u> name </u>.

07. Br tag - it is used to create line breaks in an html document.

eg.<br>.

08. Big and Small tag - with the help of this we can make the text bit larger and a bit smaller using big and small tag.

for Big - eg.<big> content </big>.
for Small - eg.<small> content </small>.

09. Hr tag - it is use to creat a horizontal line to seperate the content.

eg.<hr>.

10. Subscript and Superscript -

    Subscript - whenever we want to write any chemical formula like CO2 then we use subscript.
eg.<sub> content </sub>
code eg. to write CO2 - <p>CO<sub>2</sub></p>.
    Superscript - whenever we write anything as the power of number or formula like a2-b2 then we use Superscript.
eg.<sup> content </sup>.
code eg. to write a2-b2 - <p>a<sup>2</sup>-b<sup>2</sup>.

11. Pre tag - html always ignore extra spaces and new lines. In order to display a piece of text or any paragraph as it is, then use Pre tag.
eg.

    <pre>
      content1
      content2
      content3 ......
    </pre>
